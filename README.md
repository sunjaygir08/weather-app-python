# SkyCast Weather

SkyCast Weather is a lightweight weather dashboard built with React, TypeScript, and Open-Meteo. Search for any city, view current conditions, and browse the 7-day forecast.

## Features

1. City search with live suggestions.
2. Current weather conditions and key metrics.
3. 7-day forecast with simple weather summaries.
4. Geolocation-based first load when the browser allows it.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the app:
   `npm run dev`
3. Open the local URL shown in the terminal.

The app uses the free Open-Meteo API and does not require an API key.
